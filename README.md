Weather App
A simple and responsive Weather Application that allows users to check real-time weather information for any city using a weather API. This project is built using HTML, CSS, and JavaScript and fetches live data from a public weather API.

Features
. Search weather by city name

. Displays temperature, humidity, and wind speed

. Shows weather conditions (Clear, Cloudy, Rain, etc.)

. Responsive design (works on desktop & mobile)

. Real-time data using Weather API

Technologies Used
. HTML5 – Structure of the application

. CSS3 – Styling and responsive layout

. JavaScript (ES6) – Logic & API integration

. Weather API – For fetching real-time weather data
